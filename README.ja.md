## FuelFare: 旅費計算機とは?

**FuelFare: 旅費計算機**は [lapnito.cz s.r.o.](https://lapnito.cz) が開発した無料アプリで、あらゆる旅行の燃料費と運転コストを見積もります。距離、燃料消費量、燃料価格を入力すると、FuelFare が即座に総コストを計算します。同乗者間で割り勘、通行料や駐車料金などの追加費用を加算、車両と経路を繰り返し使用のために保存できます。

[Google Play](https://play.google.com/store/apps/details?id=com.tomas.fuelfare) と [App Store](https://apps.apple.com/us/app/fuelfare-trip-cost-calculator/id6450358923) で **FuelFare: Trip Cost Calculator** として提供されています。

ロードトリップを計画中でも、毎日の通勤費用を見積もっていても、相乗りの仲間とガソリン代を分担していても、長距離運転の予算を立てていても — FuelFare は数秒でクリアで透明性の高いコスト内訳を提供します。アプリは完全オフラインで動作し、ユーザーデータを一切収集せず、広告、サブスクリプション、アプリ内課金はありません。

## 主な機能

### 旅費計算

- **瞬時のコスト見積もり** — 距離、燃料消費量（L/100km または MPG）、燃料価格を入力すると、数秒で正確な旅費が得られます。
- **追加費用の追跡** — 燃料費に加えて通行料、駐車料、その他の運転費用を追加し、完全な旅行予算を作成します。
- **計算を表示** — 「計算を表示」をタップすると、すべてのステップが説明された完全な計算内訳が表示され、結果を確認し、各数値がどこから来たかを正確に理解できます。

### コスト分担

- **ガソリン代分割** — 2 人以上の同乗者間で燃料費と旅行費を均等に分割。相乗り、友人とのロードトリップ、家族旅行に最適。
- **一人当たりの内訳** — 各同乗者がいくら払うべきかを、燃料の分担分と追加費用を含めて一目で確認。

### 車両と経路の管理

- **車両プロファイル保存** — 車の燃料消費量と燃料価格を保存し、毎回再入力する必要をなくします。ワンタップ計算のためにデフォルト車両を設定。
- **経路保存** — 距離と追加費用が事前入力された頻繁に運転する経路（毎日の通勤、週末の旅行など）を保存。
- **旅行履歴** — デバイスにローカルに保存された過去の計算と旅行記録を確認。

### プライバシーとユーザー体験

- **100% オフライン** — 主要機能はインターネット接続なしで動作。ネットワーク呼び出しなし、クラウド依存なし。
- **広告なし、追跡なし** — 広告ゼロ、解析ゼロ、データ収集ゼロ。運転データはデバイスに留まります。
- **アカウント不要** — すぐにアプリを使い始められます。登録なし、ログインなし、メール確認なし。
- **バックアップと復元** — 車両、経路、旅行履歴を ZIP ファイルとしてエクスポート。新しいデバイスにインポートしたり、バックアップとして保持。
- **20 言語** — 日本語、英語、スペイン語、ポルトガル語、ドイツ語、フランス語、イタリア語、オランダ語、ポーランド語、チェコ語、ウクライナ語、ロシア語、トルコ語、アラビア語、ヒンディー語、簡体中国語、韓国語、インドネシア語、ベトナム語、タイ語で利用可能。

## 活用シーン

| シーン | FuelFare の役立ち方 |
|--------|---------------------|
| **ロードトリップ計画** | 長距離運転前に総燃料費を計算。通行料と駐車料を加えて完全な旅行予算を取得。友人間でコストを分担。 |
| **毎日の通勤見積もり** | 通勤経路と車両を保存し、毎日または毎週の運転にいくら燃料費がかかるかを即座に確認。 |
| **相乗りコスト分担** | 同乗者間でガソリン代を公平に分担。FuelFare が各人の燃料分担分と追加費用を計算。 |
| **車両管理と出張** | 出張の運転費用を見積もり。経費報告用に旅行記録をエクスポート。 |
| **経路比較** | 複数の経路を保存し、定期的な運転のために最も経済的な選択肢を見つけるために燃料費を比較。 |
| **予算と家計** | 旅行履歴を確認して月々の燃料費を追跡。燃料価格の変動が予算にどう影響するかを理解。 |
| **レンタカーの旅** | レンタカーの燃料消費量を入力し、借りる前に旅費を見積もり。給油所での驚きを避けられます。 |

## 旅費計算の仕組み

FuelFare は旅行の燃料費を計算するためにシンプルな式を使用します：

**燃料費 = (距離 ÷ 100) × 燃料消費量 × 燃料価格**

例えば、7.5 L/100km を消費する車で 250 km の旅行、燃料価格が 170 円/L の場合：

```
必要燃料 = 250 ÷ 100 × 7.5 = 18.75 リットル
燃料費   = 18.75 × 170 円 = 3187 円
```

通行料 500 円と駐車料 300 円を加えると、総旅費は 3987 円になります。3 人の同乗者で分割すると、一人当たり 1329 円です。

「計算を表示」機能はこの完全な内訳をステップごとに表示するので、最終数値がどのように計算されたかが常に分かります。FuelFare はメトリック（L/100km、km）とインペリアル（MPG、マイル）両方の単位をサポートしています。

**FuelFare はどれくらい正確ですか?** 計算は入力に基づいて数学的に正確です。実際のコストは、運転スタイル、交通、地形、燃料価格の変動により若干異なる場合があります。最も正確な結果を得るには、メーカーの定格値ではなく、車の実測燃料消費量を使用してください。

**電気自動車に対応していますか?** FuelFare は L/100km または MPG の燃料消費量を使用する内燃機関車両向けに設計されています。電気自動車の場合、エネルギー消費量（kWh/100km）を同等の「燃料」数値に変換する必要があります。

## よくある質問

**Q: FuelFare は無料ですか?**
A: はい、Android と iOS で完全に無料です。広告なし、サブスクリプションなし、アプリ内課金なし、アカウント不要です。

**Q: インターネットなしで動きますか?**
A: はい、FuelFare は 100% オフラインで動作します。すべての計算、保存された車両、経路、旅行履歴はデバイスにローカルに保存されます。

**Q: 同乗者とガソリン代を分担するには?**
A: いつも通り旅行の詳細を入力し、同乗者の数を設定します。FuelFare は総旅費（燃料と通行料などの追加費用）を均等に分割し、各人の分担分を表示します。

**Q: 車の燃料消費量を保存できますか?**
A: はい、燃料消費量と希望燃料価格で複数の車両プロファイルを保存できます。新しい計算で自動的に選択されるようにデフォルト車両を設定してください。

**Q: FuelFare はどの単位をサポートしていますか?**
A: メトリック（キロメートル、リットル、L/100km）とインペリアル（マイル、ガロン、MPG）両方の単位をサポートしています。アプリの設定で切り替え可能です。

**Q: FuelFare はデータを収集しますか?**
A: いいえ。FuelFare はユーザーデータを一切収集しません。解析、追跡、ネットワークリクエストはありません。すべてがデバイスに留まります。

**Q: データをバックアップできますか?**
A: はい、エクスポート機能を使用してすべての車両、経路、旅行履歴を ZIP ファイルとしてエクスポートできます。新しいデバイスにインポートするか、バックアップとして保持してください。

**Q: 「計算を表示」機能とは何ですか?**
A: 「計算を表示」は任意の旅行の完全な計算内訳を表示します — 距離と燃料消費量から最終コストまですべてのステップを表示します。結果を確認し、合計がどのように計算されたかを正確に理解するのに役立ちます。

**Q: どのデバイスがサポートされていますか?**
A: Android 7.0 (Nougat) 以降と iOS 17.0 以降。iPad でも動作します。

**Q: Android と iOS バージョンの違いは?**
A: 両バージョンとも同じ中核機能を提供します。アプリは両プラットフォームで「FuelFare: Trip Cost Calculator」と呼ばれます。
